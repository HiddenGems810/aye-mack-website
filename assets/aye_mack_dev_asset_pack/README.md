# Aye Mack Developer Asset Pack

This package contains a safe, developer-ready asset starter kit for the Aye Mack artist website.

## Important Usage Note

This ZIP intentionally does **not** redistribute copyrighted third-party photos, music videos, streaming cover art, or social media images scraped from platforms. Those assets should be pulled only after Aye Mack/client approval or from official downloadable press/EPK material.

Included instead:

- original research-derived brand direction assets
- palette images and SVG patterns
- placeholder hero/release/EPK graphics
- user-supplied social intro screenshot
- official link matrix
- music catalog seed data
- content model
- SEO keywords
- developer handoff notes
- client asset request checklist

## Suggested Folder Use

- `/assets/brand` → safe design-system images/SVGs for the developer
- `/assets/raw_user_supplied` → screenshot provided by client/user
- `/data` → JSON/CSV seed data for content modeling
- `/docs` → strategy, rights, and implementation notes
