# Client Asset Request Checklist

Send this to Aye Mack before the final build.

## Must-Have
- Official logo or approved name treatment
- 5–10 high-resolution press photos
- Latest release cover art
- Full official streaming links
- Top 5 songs to promote
- Top 5 videos to feature
- Booking/features email confirmation
- Short bio and long bio approval
- Photo/video usage permission
- Preferred domain name

## Nice-to-Have
- Show dates / event flyers
- Merch photos
- Studio BTS photos
- Lyrics/quote lines approved for web use
- Press quotes
- Management/label contact if applicable

## Rights Confirmation
For each image/video asset, confirm:
- creator/photographer credit
- usage permission for website/social ads
- commercial usage allowed or not
- whether the asset can be cropped/edited
