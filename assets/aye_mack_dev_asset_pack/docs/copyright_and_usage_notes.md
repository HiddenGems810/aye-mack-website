# Copyright / Usage Notes

Do not scrape and ship Instagram, Facebook, YouTube, Apple Music, Spotify, or press images as website assets unless the artist/client owns them or has explicit permission.

Recommended workflow:
1. Use this pack's placeholder visuals for layout development.
2. Ask Aye Mack for original press photos, official cover art, and approved logos.
3. Replace placeholders with approved assets.
4. Keep image credits in the CMS.
5. Store each image's usage rights in `GalleryImage.usageRights`.

The included brand graphics are original concept assets created for development direction and can be used as starting visual material.
